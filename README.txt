# Crypto Search Mobile App (v2 with Enhanced UI/UX)

## Requirements
- Linux or WSL
- Python 3.8+
- Java JDK
- Kivy
- Buildozer

## Build Instructions
1. Setup virtualenv and activate it
2. Install dependencies: kivy, buildozer
3. Run: buildozer -v android debug
4. APK will be in ./bin/

Includes dark mode, progress bar, result export
